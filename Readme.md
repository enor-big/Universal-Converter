# 🧮 Universal Converter

**Universal Converter** — это высокопроизводительное десктопное приложение на **Qt**, предназначенное для точного перевода неотрицательных рациональных чисел между позиционными системами счисления с основаниями от **2 до 500**.

> Проект полностью исключает использование типов с плавающей запятой (`float`, `double`, `long double`), опираясь на собственную реализацию длинной арифметики, что гарантирует абсолютную точность вычислений.

---

## ✨ Основные возможности

* **⚡ Высокая гибкость:** поддержка оснований от **2** до **500**.
* **🔢 Работа с типами:** полная поддержка целых чисел, конечных и периодических дробей.
* **🧩 Универсальность ввода:** поддержка стандартных цифр (`0-9`, `A-Z`) и расширенного формата `[number]`.
* **⚙️ Собственная математика:** реализация `BigInteger` (база $10^9$) и `BigFraction`.
* **🚀 Оптимизация:** встроенный **алгоритм Карацубы** для быстрого умножения больших чисел.
* **📂 Инструментарий:** проверка данных, загрузка и сохранение результатов в файл.

---

## 📋 Формат ввода

Программа принимает данные в строгом порядке:

1.  **`p`** — основание исходной системы счисления.
2.  **`q`** — основание целевой системы счисления.
3.  **`input`** — число для конвертации.

### Примеры поддерживаемых форматов

| Тип | Примеры |
| :--- | :--- |
| **Целые** | `123`, `A1`, `[222][223]` |
| **Конечные дроби** | `123.45`, `A.F`, `0.[10][20]` |
| **Периодические дроби** | `0.(3)`, `0.12(34)`, `1.A(B)`, `0.[10][20]([30])` |

---

## 🏗 Архитектура проекта

Проект спроектирован с учетом принципа разделения ответственности:

* `MainWindow` — графический интерфейс (Qt Widgets).
* `BaseConverter` — главный контроллер логики.
* `BaseParser` — парсинг и валидация строки.
* `BigFraction` — хранение дробей (`numerator` / `denominator`).
* `BigInteger` — ядро длинной арифметики.
* `BaseFormatter` — преобразование в целевую систему.

---

## 🛠 Сборка и запуск

1.  Откройте файл `UniversalConverter.pro` в **Qt Creator**.
2.  Выберите нужный Kit (требуется **C++17**).
3.  Нажмите кнопку **Build** (или `Ctrl+B`).
4.  Запустите приложение.

---

## ⚠️ Ограничения
* **Только неотрицательные числа.**
* **Максимальная длина вывода:** `100 000` символов.
* **Сторонние библиотеки:** использование внешних библиотек для длинной арифметики запрещено.
