\# Universal Converter



\*\*Universal Converter\*\* — это высокопроизводительное оконное приложение на \*\*Qt\*\*, предназначенное для точного перевода неотрицательных рациональных чисел между позиционными системами счисления с основаниями от \*\*2 до 500\*\*.



Проект полностью исключает использование типов с плавающей запятой (`float`, `double`, `long double`), опираясь на собственную архитектуру длинной арифметики, что гарантирует абсолютную точность вычислений.



\---



\## 🚀 Основные возможности



\*   \*\*Гибкость:\*\* работа с основаниями от \*\*2\*\* до \*\*500\*\*.

\*   \*\*Типы данных:\*\* поддержка целых чисел, конечных и периодических дробей.

\*   \*\*Универсальность ввода:\*\* поддержка цифр `0-9`, `A-Z` и формата `\[number]`.

\*   \*\*Длинная арифметика:\*\* собственная реализация `BigInteger` (база $10^9$) и `BigFraction`.

\*   \*\*Оптимизация:\*\* использование \*\*алгоритма Карацубы\*\* для ускорения умножения больших чисел.

\*   \*\*Инструментарий:\*\* проверка корректности ввода, загрузка и сохранение данных через файлы.



\---



\## 📋 Формат ввода



Программа обрабатывает данные в следующем порядке:

1\. \*\*`p`\*\* — основание исходной системы счисления.

2\. \*\*`q`\*\* — основание целевой системы счисления.

3\. \*\*`input`\*\* — число для конвертации.



\### Примеры поддерживаемых чисел



| Тип | Примеры |

| :--- | :--- |

| \*\*Целые\*\* | `123`, `A1`, `\[222]\[223]` |

| \*\*Конечные дроби\*\* | `123.45`, `A.F`, `0.\[10]\[20]` |

| \*\*Периодические дроби\*\* | `0.(3)`, `0.12(34)`, `1.A(B)`, `0.\[10]\[20](\[30])` |



\---



\## 🏗 Архитектура приложения



Проект построен на принципах модульности:



\*   \*\*`MainWindow`\*\*: Пользовательский интерфейс (Qt Widgets).

\*   \*\*`BaseConverter`\*\*: Центральный узел логики конвертации.

\*   \*\*`BaseParser`\*\*: Валидация и парсинг входных данных в `BigFraction`.

\*   \*\*`BigFraction`\*\*: Хранение дробей через два объекта `BigInteger`.

\*   \*\*`BigInteger`\*\*: Ядро арифметических операций (сложение, вычитание, умножение, деление, GCD).

\*   \*\*`BaseFormatter`\*\*: Форматирование результата для целевой системы счисления.

\## 🛠 Сборка и запуск



1\. Откройте файл `UniversalConverter.pro` в \*\*Qt Creator\*\*.

2\. Настройте необходимый \*\*Kit\*\* (проект требует \*\*C++17\*\*).

3\. Нажмите \*\*Build\*\* (сборка) и запустите приложение.



> \*\*Примечание:\*\* Для работы проекта не требуются внешние библиотеки, всё реализовано на стандартном C++ и Qt.



\---



\## ⚠️ Ограничения



\*   \*\*Числа:\*\* проект работает только с \*\*неотрицательными\*\* рациональными числами.

\*   \*\*Лимит вывода:\*\* максимальная длина результата ограничена `100 000` символами.

\*   \*\*Зависимости:\*\* использование сторонних библиотек для длинной арифметики \*\*запрещено\*\*.
